# Financial Fraud Detection System

Fraud detection platform using Machine Learning and anomaly detection.

## Features

- Fraud detection
- Isolation Forest model
- Streamlit dashboard
- Financial transaction analysis
- Interactive charts

## Technologies

- Python
- Pandas
- Scikit-Learn
- Plotly
- Streamlit

## Installation

```bash
pip install -r requirements.txt
```

Train model:

```bash
python train_model.py
```

Run dashboard:

```bash
streamlit run app.py
```
